# qover-car-insurance

### Prerequisites
- nodejs (v16)
- mongodb
- docker

### Usage

1. Clone repo  
2. Install dependencies in each folder (backend && frontend)
```sh
$ npm install
```
3. By preference, you can insert default data in db by running a script
```sh
$ npm run injectData
```
4. Start server
```sh
qover-car-insurance/backend/
$ npm run start
```
5. Start client
```sh
qover-car-insurance/frontend/
$ npm start
```
